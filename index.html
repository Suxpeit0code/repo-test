Python

Python é uma linguagem de programação de alto nível, conhecida por sua sintaxe simples e legível. É amplamente utilizada em desenvolvimento web, automação, ciência de dados, inteligência artificial e aprendizado de máquina.

JavaScript

JavaScript é a linguagem principal para desenvolvimento web e é executada nos navegadores dos usuários. É usada para criar interações dinâmicas em páginas web, desenvolvimento de front-end e também é usada no desenvolvimento back-end (Node.js).

Java

Java é uma linguagem de programação popular que é conhecida por sua portabilidade e é amplamente usada no desenvolvimento de aplicativos corporativos, aplicativos Android e sistemas de grande escala.

C#

Desenvolvida pela Microsoft, C# é usada principalmente para desenvolvimento na plataforma .NET. É amplamente utilizada para criar aplicativos Windows, jogos e soluções corporativas.

C++

Uma linguagem de programação de propósito geral, que é amplamente usada em sistemas operacionais, aplicativos de desktop, jogos, software embarcado e outras aplicações de desempenho crítico.

PHP

PHP é uma linguagem de programação amplamente usada para desenvolvimento web do lado do servidor. É especialmente popular para criação de sites dinâmicos e aplicativos web.

TypeScript

Uma extensão do JavaScript que adiciona recursos de tipagem estática. É usado principalmente em projetos grandes e complexos para evitar erros comuns de tipagem

Ruby

Ruby é conhecida por sua sintaxe amigável e é frequentemente usada no desenvolvimento web, especialmente em conjunto com o framework Ruby on Rails.

Swift

Desenvolvida pela Apple, Swift é a linguagem principal para o desenvolvimento de aplicativos iOS e macOS.

Go

Também conhecida como Golang, Go é uma linguagem desenvolvida pelo Google. É popular para desenvolvimento de aplicativos de rede e sistemas distribuídos.
